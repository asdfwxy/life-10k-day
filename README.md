# ⏳ 人生一万天 (Life 10k Days)

> *“如果把一生按80年计算，我们大约拥有三万天。<br>但抛去睡眠、生存与责任，真正属于你的时间，不到一万天。”*

「人生一万天」是一款基于纯原生 (Vanilla) 技术构建的极简、私密、富有纸质哲理美学的云端日记与文化推荐 Web 应用。它不提醒你要焦虑逝去的光阴，而是温和地询问你：**在这有限的时间里，你想把多少留给自己？**

---

## ✨ 核心特色功能 (Features)

### 🪶 纸质美学与 3D 悬浮交互
- 摒弃现代繁杂的 UI 框架，采用深思熟虑的微黄纸张质感与全屏 SVG 噪点覆盖。
- 流畅的 **3D 翻转卡片 (Flip Card)**：正面书写心绪，回车翻面即能遇见与你灵魂共振的书籍、电影与音乐推荐。
- 细腻的鼠标光标追踪微动特效，随着视角的轻微移动，赋予网页真实的呼吸感。

### ☁️ 云端灵魂绑定 (Firebase Integration)
- 接入了 **Google OAuth 一键登入**：无需繁杂注册过程。
- 接入了 **Cloud Firestore**：所有日子与沉思皆持久化在属于你的云端，换电脑、清空记录也无需担心丢失。
- 本地历史记录自动无痕迁移至云端机制。

### 🗓️ 时空热力网格图
- 借鉴了 GitHub 贡献绿格子设计的“时光抽屉”，采用极简右侧拉开式动画。
- 按照真实的生命跨度（从出生计算）绘制日历，可以**直达未来，亦可拨回过去**。
- **补记机制**：点击日历中任意缺失的过去日子，打卡卡片便能穿越时空，显示着真实的“过去日期”供你重构当时的回忆。

### ⛅ 自动天气记忆系统
- 对接世界天气服务 API，根据用户的地理 IP，自动关联当天的天气符号和高低气温。
- **双模引擎架构**：补写过去5天内的日子调用最新 Forecast，过去多年的日子自动调用 Archive 档案气象，让过去的记忆里有风也有雨。

## 🛠️ 技术引擎 (Technology Stack)

在框架横行的时代，本项目坚持采用绝对零依赖方式，只为了极致纯粹与零维护成本：
- **核心构建**：纯原生 HTML5, CSS3 变量与 Flex/Grid 布局, 原生 JavaScript ES6。
- **云端服务**：Firebase 10.x SDK (通过原生浏览器 ES Module 直接 CDN 引入，免除一切打包工具如 Webpack/Vite)。

## 🚀 本地运行方式 (How to run locally)

由于应用使用了模块化（`type="module"`）引入 Firebase JS SDK，为避开现代浏览器的本地 CORS 严格跨域限制，你只需要一个简单的本地微型服务器：

1. 克隆或下载此仓库内容至电脑。
2. 在终端（Terminal）进入该目录，运行：
```bash
npx serve .
```
*(或者如果你习惯使用 Python：)*
```bash
python3 -m http.server 3000
```
3. 在浏览器打开 `http://localhost:3000` 即可开始丈量你生命的时钟。
